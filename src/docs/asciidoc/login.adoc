어플리케이션에 로그인합니다.
만약 존재하지 않는 아이디(이메일)이거나, 비밀번호가 틀릴 경우 실패합니다.
OAuth Login 과는 다릅니다.
OAuth Login API에 대해서 알고 싶다면 <<oauth>> 를 참고해주시기 바랍니다

=== Request
Request URL::
[source,http,options="nowrap"]
----
POST /users/login
----

Request Headers::
[source,http,options="nowrap"]
----
Content-Type: application/json;charset=UTF-8
----

Request Body::

[source,http,options="nowrap"]
----
{
    "email" : String,
    "password" : String,
    "confirmPassword" : String
}
----
