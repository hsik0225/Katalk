
어플리케이션에 신규 유저를 등록합니다.

IMPORTANT: 이미 가입된 ID(Email)로 회원가입을 진행하실 경우 회원가입에 실패합니다

=== Request

==== Request URL

[source,http,options="nowrap"]
----
POST /users
----

==== Request Headers

[source,http,options="nowrap"]
----
Content-Type: application/json;charset=UTF-8
----

==== Request Body

[source,http,options="nowrap"]
----
{
  "signManager" : {
    "email" : String,
    "password" : String,
    "confirmPassword" : String
  },
  "profileManager" : {
    "chatName" : String,
    "picture" : String
  }
}
----
