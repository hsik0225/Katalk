== Common

API에 관계없이 아래 사항을 지켜주셔야 합니다.

== Domain

|===
| 환경 | domain

| 개발 서버
| `dev.localhost`

| 운영 서버
| `real.localhost`
|===

=== Header

|===
| name | description

| `Authorization`
| API를 사용하기 위한 인증 키
|===

=== 공통 Response Body

|===
| field | description

| `code`
| 응답 코드

| `message`
| 예외사항 내용 등 응답메세지

| `data`
| 실제 반환될 데이터
|===
